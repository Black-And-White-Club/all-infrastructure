### cert-manager v1.16.2 → v1.20.2 — Upgrade Notes (4 minor versions)

This is a significant jump across four minor releases. Review the upstream upgrade guide before merging:
<https://cert-manager.io/docs/installation/upgrade/>

**v1.18 breaking change:** The `PrivateKey.RotationPolicy` feature gate is now GA in v1.19 and cannot be disabled. Before upgrading through v1.18, ensure all Certificate resources have an explicit `rotationPolicy` set (e.g. `rotationPolicy: Never`) or accept the new default behaviour.

**v1.19 metric label change:** The high-cardinality `path` label was removed from `certmanager_acme_client_request_count` and `certmanager_acme_client_request_duration_seconds` metrics; a new `action` label replaces it. Update any dashboards or alerting rules that reference the old label.

**v1.20 Helm fix:** Fixes invalid YAML generated when both `webhook.config` and `webhook.volumes` are defined.
