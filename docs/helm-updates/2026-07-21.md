The file contains a hard warning: *"PINNED: Do NOT bump this version without verifying the

- Chart 16.2.4 (currently pinned) ships **PostgreSQL 17.2.0** (confirmed via `Chart.yaml` tag).
- The latest chart on the bitnami/charts GitHub main branch is **17.1.0** with **PG 17.6.0** —

  16.2.4→17.1.0 in terms of initdb. However, going from chart `16.x` → `17.x` IS a Helm chart
  **major** version bump with potentially breaking values changes.
- Bitnami primarily distributes via OCI (`oci://registry-1.docker.io/bitnamicharts`) and the
  GitHub repo may lag behind actual releases. The true latest version cannot be confirmed without
  Helm access to bitnami's registry (blocked in this environment).

**This chart must be updated manually** after verifying:
1. The latest chart version (run `helm search repo bitnami/postgresql --versions` locally).
2. That the `appVersion` (PostgreSQL image tag) stays on the same major version.
3. Any values changes in the 17.x chart schema vs. `charts/postgres/frolf-values.yaml`.
