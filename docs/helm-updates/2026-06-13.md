## Version Discovery Method

Helm repository HTTP traffic is blocked by the network policy in the CI environment. Chart versions
were sourced using `git ls-remote --tags` against each chart's upstream GitHub repository, which
uses the git protocol and is permitted. Versions were parsed and sorted using Python's `packaging`
library. The GitHub REST API (for release notes) was exhausted (0/60 unauthenticated requests
remaining) for the duration of the run.

For the next run, adding a `GITHUB_TOKEN` environment variable to the CI environment would enable
authenticated GitHub API calls (5000 req/hour), allowing automated changelog fetching.
